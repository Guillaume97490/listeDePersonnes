# listeDePersonnes

## Consigne : 
Afficher dans un tableau la liste des entrée de la base sur la page visiteur voir un élément au hasard avec un random valide 1 jour.

## Structure de la BDD
- nom (varchar)
- prénom (varchar)
- photo (varchar)
- domaine (varchar)
- DoB (datetime)
- ville (varchar)
- genre (boolean)
- dateChoisi (datetime)
